당담기능 세부설명
  • 공지사항
    • Restful API를 활용한 CRUD 방식의 공지사항 구현. CKEditor API를 활용하여 게시글작성 및 수정 구현.
    • Spring security 를 활용하여 게시글 작성, 수정, 삭제 가능한 관리자와 조회만 가능한 직원으로 권한 제한.
  • 쪽지
    • 쪽지보내기- 직원검색 모달 창에서 부서, 팀, 이름을 조회하여 AJAX 비동기 방식으로 직원 이름을 반환.
    • 휴지통(message_bin) 상태값을 0(보관함)과 1(휴지통)으로 나누어 복구 기능 구현.
    • senderEmpNo(발신자), recipientEmpNo(수신자) 컬럼으로 나눈 뒤 조회, 답장 기능 구현.
  • 휴가관리
    • 결재문서(document 칼럼)에서 '문서유형: 휴가 + 승인상태: 승인' 된 문서를 조회하여 카운팅.
    • 휴가유형(vacation_type)을 연차, 반차(오전/오후)로 나누어 휴가 관리.
    • 기존 조회 테이블을 js에서 hidden 처리한 뒤, 시작일과 종료일을 기준으로
      필터링하여 AJAX 로 반환된 새로운 테이블 생성.
  • 근태조회
    • 메인화면에서 if 문과 Date 함수를 사용하여 출근 & 퇴근 기능 구현.
    • 결재문서(휴가유형, 승인 컬럼) 테이블과 근태 테이블을 조인하여 출근, 결근, 반차, 연차 직원 값 반환.
    • AJAX로 부서, 팀, 이름으로 직원 검색 시 근태(attendance) 테이블을 조회하여 js 동적테이블 생성.
